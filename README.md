
# 🚀 UTM Manager Outlook Add-in

UTM Manager is a powerful Outlook Add-in that automatically generates and inserts UTM-tracked links into your email signature. It helps marketing and sales teams track email campaign performance directly from Outlook.

## ✨ Features

- 🔗 Automatically generate UTM-tagged links  
- 📩 Insert links into Outlook email signatures

## ⚙️ How It Works

1. Open Outlook and start composing an email  
2. Launch the UTM Manager Add-in  
3. Enter campaign details (source, medium, campaign, etc.)  
4. Generate a UTM link  
5. Automatically insert it into your email signature  
6. Track clicks in analytics tools like Google Analytics  

---

### 5. Sideload Add-in into Outlook

* Open Outlook
* Go to Add-ins → Upload My Add-in
* Upload the manifest file
